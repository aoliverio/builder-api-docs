Builder Core

Builder is core plugin, developed for CakePHP 3.x application, that implements a RAD toolkit to building admin web applications in Bootstrap style.

  • Releases: [1.0]

Builder Applications

All CakePHP application developed using the Builder enviroment.

  • Builder SterterKit (under construction).
  • Builder DemoApp (under construction).

Builder Extensions

All CakePHP plugin developed to using the Builder enviroment.

  • Content, provide a content management system backend.
  • Site, provide a content management system frontend.
